Portfolio Details

This dataset focuses on the prediction of indicators/diagnosis of diabetes.

The features cover contains the distribution for 70 sets of data recorded on diabetes patients (several weeks' to months' worth of glucose, insulin, and lifestyle data per patient + a description of the problem domain).

Project information

  • Title: Risk Factors in Diabetes
  • Owner: Dylan
  • Project date: 01 March, 2021
  • Project URL: GitHub

Project information

  • Characteristics: Multivariate with 70 number of instances
  • Technique: Classification
  • Hyperparameters: Tabular, 12 batch size, 3/10 Learning Rates, 25 Number of Epochs
  • Accuracy: 90%
  • Appendices: RID(Regular insulin dose), BGM(blood glucose measurement), Age, HS(Hypoglycemic symptoms)
