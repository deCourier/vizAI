Portfolio Details

This dataset focuses on the prediction of age/height/weight distributions of NBA players.

The data set contains over two decades of data on each player who has been part of an NBA teams' roster. It captures demographic variables such as age, height, weight and place of birth, biographical details like the team played for, draft year and round. In addition, it has basic box score statistics such as games played, average number of points, rebounds, assists, etc. The pull initially contained 52 rows of missing data. The gaps have been manually filled using data from Basketball Reference. I am not aware of any other data quality issues.

Project information

  • Title: Ideal NBA Player Physical Characteristics
  • Owner: Dylan
  • Project date: 01 March, 2021
  • Project URL: GitHub

Project information

  • Characteristics: Multivariate with 2235 number of instances
  • Technique: Classification
  • Hyperparameters: Tabular, 32 batch size, 6/10 Learning Rates, 25 Number of Epochs
  • Accuracy: 90%
  • Appendices: PPG(Points Per Game), MPG(Minutes Per Game), Age, RPG(Rebounds Per Game)
