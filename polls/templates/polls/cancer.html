Portfolio Details

This dataset focuses on the prediction of indicators/diagnosis of cervical cancer.

The features cover demographic information, habits, and historic medical records. The dataset was collected at 'Hospital Universitario de Caracas' in Caracas, Venezuela. The dataset comprises demographic information, habits, and historic medical records of 858 patients.

Project information

  • Title: Risk Factors in Cervical Cancer
  • Owner: Dylan
  • Project date: 01 March, 2021
  • Project URL: GitHub

Project information

  • Characteristics: Multivariate with 858 number of instances
  • Technique: Classification
  • Hyperparameters: Tabular, 32 batch size, 3/10 Learning Rates, 25 Number of Epochs
  • Accuracy: 85%
