Related Projects Review

Unfortunately there is no similar project to the VizAI project. In general, machine learning cloud computation services have built in explainability modules, but those modules are proprietary tools that can only be used on models and datasets trained and tested within the framework of the service provider. 

However, the team did manage to find useful information and features from Microsoft Azure's machine learning platform:

NameAzure Machine Learning
Main featuresAutomated ML, Explainability module
What can be learnedThe importance of intuitive and logical graph presentation when explaining a machine learning model
Automated ML

In Azure Machine Learning's "Automated ML" section, a user selects a dataset and the desired column of values to predict, then starts running the module on a remote compute target, and Automated ML module will automatically find the best algorithm for training the output model and explain it:

Explainability Module

The explainability module is able to display the explanation and relating graphs intuitively. Below is an explanation on a trained model to predict the total amount of alcohol consumed annually by countries around the world:

As seen, out of the three types of alcohols served for each country, the amount of beer served annually contributes the most to a nation's annual consumption of all alcohol, while each of wine and other spirits have only about half the contribution.

From here it can be easily deduced that when predicting the annual consumption of alcohol of a country, the model's prediction is affected more by the amount of beer that country serves, making the prediction pattern logical and understandable.
